Why does this website look like this? Is it broken?

This look is intentional. I thought long and hard about how I would make my personal website. I could've definetly whipped up some css to make it look cool but, as you might have guessed from my tech stack, I mostly focus on backend. I thought this might be effective enough in scaring away recruiters trying to find somebody to work on their fad JavaScript framework website.

And, as mfw reminds us, this kind of website checks all the usual boxes:

  • It's responsive.
  • It's fast and light.
  • It will work on any device (and any browser).
  • It's easily indexed.
